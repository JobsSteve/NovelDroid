NovelDroid
==========
An android app for extracting novel contents from forum posts

##Latest Version
1.1.5

##Supporting Sites
* cklol
* 3yny

##Features
1. Strong Analyzer: parse book name and author almost perfectly
2. Settings: encoding, filename, output folder
3. Search: with google custom search
